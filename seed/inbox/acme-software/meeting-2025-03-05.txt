DEMO NOTES

Sarah: "We are fully multi-tenant with strict data isolation — separate schemas per tenant, no shared compute. I'll cover this in the technical spec. If David wants to get into the architecture in detail, I can also set up a call with our solutions engineer."

James: "Let's do that — get us the tech spec first and then we'll decide if we need the solutions engineer call."
